# New Peddy Website

## Project Description
- **New peddy** is a JavaScript-based web project designed to provide a smooth  
 and user-friendly experience for pet-related services.

 - **New peddy** is a JavaScript-based web application that makes pet adoption  
  and management easier with a simple and interactive user interface.

  - **New peddy** is a JavaScript project built to create a simple and interactive  
   pet-focused web experience.
   ---


   ## 5 key features this project

   1. Browse and view available pets with detailed information  
   2. User-friendly and responsive interface  
   3. Interactive UI built with pure JavaScript 
   4. Simple and clean design for easy navigation  
   5. ast and lightweight web experience
